# HuXa Color Palette

These are the canonical colors used across all platforms.

| Name       | Hex       | Usage                              |
|------------|-----------|-------------------------------------|
| Background | `#1a1a2e` | App background, icon background     |
| Surface    | `#16213e` | Cards, elevated surfaces            |
| Input      | `#0f3460` | Input fields, deep accents          |
| Text       | `#e0e0e0` | Primary text                        |
| Muted      | `#8a8a9a` | Secondary text, borders             |
| Accent     | `#e94560` | Primary accent, CTAs, coral red     |
| Success    | `#4ecdc4` | Success states, teal/mint highlight |
| Error      | `#ff6b6b` | Error states, destructive actions   |

## Design Notes

- The palette is dark-first — navy/midnight tones dominate
- Coral red (`#e94560`) is the signature accent — use sparingly for impact
- Teal (`#4ecdc4`) is the secondary accent — used for positive/success states
- The icon uses these colors: navy background, frosted glass brain in muted blues, coral neural pathway, teal connection nodes
